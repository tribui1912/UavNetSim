UavNetSim Launchers - Quick Summary
====================================

WINDOWS USERS:
  Double-click: launcher\run_uavnetsim.bat

MAC/LINUX USERS:
  Run: ./launcher/run_uavnetsim.sh
  (First time: chmod +x launcher/*.sh)

WHAT IT DOES:
  - Checks Python & uv
  - Sets up virtual environment
  - Installs dependencies
  - Shows interactive menu

MENU OPTIONS:
  [1] Simulation
  [2] All Tests
  [3] Individual Test
  [4] All Experiments
  [5] Individual Experiment
  [6] System Status
  [7] Update Deps
  [8] Quick Ref
  [9] Exit

FILES INCLUDED:
  - run_uavnetsim.bat/.sh   (Full menu)
  - quick_start.bat/.sh     (Quick sim)
  - README.md               (Full docs)

NO SETUP NEEDED - JUST RUN IT!
